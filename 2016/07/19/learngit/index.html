git

创建版本库

git init 把这个目录变成Git可以管理的仓库
git add 把文件添加到仓库
git commit -m”更改说明” 把文件提交到仓库
git status 查看工作区的状态
如果git status告诉你有文件被修改过，用git diff可以查看修改内容。
git log 告诉我们历史记录
git reset 版本回退

上一个版本，HEAD当前版本。

git reflog 记录每一次命令
