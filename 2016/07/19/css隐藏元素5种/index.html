CSS隐藏页面元素6种

Opacity

元素透明度设为0,只是视觉上不可见,元素仍占据自己的位置.
对于读屏软件无用,内容还是会显示.

Visibility

设为hidden,同opacity,隐藏元素依旧会对网页布局起作用
注意:如果一个元素的 visibility 被设置为 hidden，同时想要显示它的某个子孙元素，只要将那个元素的 visibility 显式设置为 visible 即可

Display

没有交互效果,不占据位置,子元素也会隐藏,DOM依然可以访问.

Position

既可以交互,又不影响布局

你得避免使用这个方法去隐藏任何可以获得焦点的元素，因为如果那么做，当用户让那个元素获得焦点时，会导致一个不可预料的焦点切换。这个方法在创建自定义复选框和单选按钮时经常被使用。

Clip-path

通过裁减元素实现,clip-path 属性还没有在 IE 或者 Edge 下被完全支持

text-indent

文本缩进,一般用于隐藏文字
通过使文字溢出但不显示（overflow:hidden）的方式来达到隐藏文字的
